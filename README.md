Deskripsi project (fungsi tiap halaman):
Project ini adalah sebuah aplikasi berbasis web sederhana untuk manajemen produk yang membuatnya menggunakan HTML,CSS, dan JavaScript. Aplikasi ini memiliki fitur autentikasi(login) dan juga sistem CRUD(Create,Read,Update,Delete) untuk mengelola data produknya.
a. Fungsi Halaman Login: Halaman ini berguna untuk user memasukkan email dan password. Untuk email dan password tidak boleh kosong dan jika validasi input yang dimasukkan benar user akan diarahkan ke halaman Dashboard.
b. Fungsi Halaman Dashboard: Ini adalah halaman utama setelah berhasil login. Yang menampilkan ringkasan data dalam bentuk card ada total products(jumlah produk yang ada), total sales(jumlah penjualan), total revenue(total pendapatan). Untuk button "Lihat Data Produk" diklik maka akan diarahkan ke halaman Products.
c. Fungsi Halaman Products: Halaman ini menampilkan tabel daftar produk dengan kolom no, product name, price, stock, aksi. Dimana user bisa melihat,mengedit dan juga menghapus produk.
